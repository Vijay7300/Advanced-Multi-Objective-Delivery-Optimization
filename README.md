# 🚚 Advanced Multi-Objective Delivery Optimization System

This project focuses on solving a real-world multi-objective optimization problem in delivery and logistics systems using both deterministic optimization techniques and heuristic algorithms. The system aims to optimize delivery operations by minimizing transportation cost, delivery time, and operational risk while improving efficiency and resource utilization. 

### 🚀 Features

* Multi-objective optimization framework
* Delivery cost and route optimization
* Risk-aware delivery planning
* Real-world dataset integration
* Genetic Algorithm based optimization
* Quadratic Programming based scalarization approach
* Data visualization and analysis
* Interactive deployment using Streamlit

### 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* PuLP

* DEAP (Genetic Algorithm Library)
* Streamlit

### 📊 Optimization Workflow

1. Real-world dataset collection
2. Data preprocessing and merging
3. Feature engineering
4. Multi-objective optimization formulation
5. Deterministic optimization
6. Genetic Algorithm implementation
7. Performance comparison
8. Visualization and deployment

### 📈 Optimization Objectives

### 📌 Dataset Information

The project uses the Brazilian E-commerce public dataset from Kaggle and creates a realistic delivery optimization dataset using:

* Freight cost
* Delivery time
* Delivery distance
* Package weight
* Delivery zones 

### 🎯 Project Goal

The primary objective of this project is to demonstrate practical implementation of multi-objective optimization techniques for logistics and delivery systems using Machine Learning, Operations Research, and Evolutionary Algorithms.
